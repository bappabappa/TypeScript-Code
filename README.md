# TypeScript-Code
### 2. Environment setup

#### 2.1 Install and use typescript

- Text Editor: VS Code
- Install node & typescript
  ```js
     npm intsall typescript --save-dev (local)
     npm install -g typescript (global)
     npx tsc fileName.ts
  ```
- check various versions:

  ```js
    node --version
    npm --version
    tsc --version
  ```

#### 2.2 How does it work?

- index.ts -> tsc index.ts -> index.js
